# Abulafia TS

Abulafia-TS is a typescript library for parsing and working with the type of tables at Abulafia / http://www.random-generator.com/

## Setup

```
$ cd ../abulafia-ts
$ npm build
```

## Run All Tests

```
$ cd ../abulafia-ts
$ npm test
```

## Unit Tests

```
$ cd ../abulafia-ts
$ npm test-unit
```

## Mutation Tests

```
$ cd ../abulafia-ts
$ npm test:mutation
```
